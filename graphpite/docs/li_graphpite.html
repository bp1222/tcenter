What is GraPHPite

GraPHPite is pronounced "graphite", as in the layered hexagonal structure of carbon, is an acronym of Graph and PHP and 'ite to make it pronouncable.

GraPHPite as is indicated by the title a PHP project to, via PHP's object oriented model, simply but very customizable enable the user/developer to get exactly the look and feel they want.

GraPHPite is still in it's early stages of development, but functionality includes:

Many chart types, amongst others:

  • Bar charts

  • Area charts

  • Line charts

  • Smoothed area charts

  • Smoothed line charts

Many types of data source:

  • Simple array based

  • Functions

  • Vector functions

  • (my)SQL (to come)

Many charts displayed in the same graph

Gradient colored filling

Every line, every color, every font can be customized

Output as PNG or JPEG

Requirements

The only system requirement is PHP 4 support with GD 2 support. GD 1.x support is currently only experimental.
