Class Details

[line 45]
Layout for displaying two elements side by side.

This splits the area contained by this element in two, side by side by a specified percentage (relative to the left side). A layout can be nested. Fx. a HorizontalLayout can layout two {@see Image_Graph_Layout_Vertical}s to make a 2 by 2 matrix of "element-areas".
